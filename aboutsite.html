This CV is NOT static.

It was built using PHP, MySQL, jQuery, Bootstrap and Angular. Most of my resume is Microsoft focused, so this acts as proof I can navigate the LAMP stack (or any other for that matter).


All the data for jobs, accomplishments, reccomendations and skills are stored first in MySQL and sent in seperate arrays via a PHP generated JSON resume object that you can inspect here. From that point, I organize the data, and dynamically display it on the page using Angular and Bootstrap. If you're hardcore, you might want to see this GitHub link which includes all the code. For now, I've focused on Desktop, but will add some mobile specific functionality soon.
