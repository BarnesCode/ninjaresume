Interactive Curriculum Vitae

This site was built using PHP, MySQL, and Angular. Most of my resume is Microsoft focused, so this acts as proof I can navigate the LAMP stack.


All the data for jobs, accomplishments and skills are stored in MySQL and sent in seperate arrays via a PHP generated JSON resume object that you can inspect here. From that point, I organize the data, and display it on the page using Angular and Bootstrap.

Because I need to share this quickly, I'm going intentionally light on design, reserving time for functionality.

My ideal position is technical co-founder, or employee #1 for a recently funded startup. I prefer a culture centered around teamwork, expirimentation, and people who are excited.

Use the navigation on the right, scroll down, or use any of the following links to jump to my Work History, Major Accomplishments, or Skills Toolbox.
